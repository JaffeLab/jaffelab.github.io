tc.ListFiles()

Syntax

tc.listFiles(folder)

Description

Lists all of the files within a particular folder.

Example

folder = '/content/gdrive/My Drive/LabData/InVivoData’
tc.listFiles(folder)

tc.makeFname()

Syntax

fname = tc.makeFname(run, folder, Exp)

Description

Generates a string based on run number, file location (folder), and experiment name.

Example

RUN = 0
Exp = ‘XXMMDDYY’ # Experiment name
mainFolder = ‘/content/gdrive/My Drive/LabData/InVivoData’
fname = ‘/content/gdrive/My Drive/LabData/InVivoData/XXMMDDYY/XXMMDDYY-0.smr’

tc.Loadfile()

Syntax

time, ch0, ch1 = tc.Loadfile(folder)

Description

Loads a run from a datafile and extracts time, channel0, and channel1 arrays.

Example

fname = mainFolder
time, ch0, ch1 = tc.Loadfile(fname)

tc.plotData()

Syntax

tc.plotData(time,ch0,ch1)

Description

Plots two channels of data arrays against time array. The optional keyword arguement pair outputName='filename' will save the plot as both a vector graphics (svg) and bitmap (jpg) file to the selected folder (%cd /content/gdrive/My\ Drive/Folder.)

Example

tc.plotData(time, ch0, ch1, ​outputName=’String’​)

tc.clip()

Syntax

time_clip, ch0_clip, ch1_clip = tc.clip(start,stop, time,ch0,ch1)

Description

Indexes a subset of points based on a start/stop times (in milliseconds) and the time, channel0, and channel1 arrays.

Example

tm_clp, c0_clp, c1_clp = tc.Loadfile(10000,11000,time,ch0,ch1)

tc.lpFilter()

Syntax

f0, f1 = tc.lpFilter(c0, c1, order, cutoff)

Description

Performs low-pass filtering of two arrays c0,c1 based on order(1-n) and cutoff frequency (Hz).

Example

f0, f1 = tc.lpFilter(c0, c1, 6, 220)

tc.PSD()

Syntax

f0, psd0, f1, psd1 = tc.PSD(c0, c1, order, **kwargs)

Description

Performs Power Spectral Density Analysis on two data arrays c0,c1 based on order(1-n).
Keyword arguement pairs: plot = 'true', xmax = n, the former generates a plot and the latter adjusts the maximum dimension of the x-axis. Return values are frequency and power arrays for both channels.

Examples

f0, psd0, f1, psd1 = tc.PSD(c0, c1, 10)
f0, psd0, f1, psd1 = tc.PSD(c0, c1, 10, plot='true')
f0, psd0, f1, psd1 = tc.PSD(c0, c1, 10, plot='true', xmax=25)
